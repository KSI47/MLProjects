
- Ci-joint 2 programmes python :
	1- Pour 6 paramètres en entrée
	2- Pour tous les 32 paramètres en entrée
- ils exploitent tous les deux les bibliothèques suivantes :
	1-sklearn
	2-matplotlib
	3-numpy
	4-pandas
	5-math
elles doivent donc etre installées avec pip (gestionnaire de paquets python)
- pour les executer il faut deplacer les fichiers train.csv et test.csv 
dans le meme répertoire que app.py et app32.py (ils le sont par défaut)

- avec Python3 , vous executer à partir d'un terminal la commande 
	1- $ cd /path-to-directory/codes_svm/
	2- $ python app.py pour le premier code
	3- $ python app32.py pour le deuxième code
